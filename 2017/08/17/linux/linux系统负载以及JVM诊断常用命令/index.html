linux系统负载以及JVM诊断常用命令

发表于 2017-08-17 | 分类于 技术文章

一些基本命令备忘

  • 查看端口号被占用情况:

    fuser -n tcp 9200

  • 查看所有进程,以及cpu,内存占用:

    ps aux

  • 查看机器内存用量以及剩余:

    free -m

  • 查看指定进程内, 线程资源占用情况:

    top -Hp pid

  • 转换pid -> nid:

    printf "%x\n" pid

  • java堆统计实例数量,大小:

    jmap -histo:live pid | head -100

  • java栈快照:

    jstack -l  nid

  • 文件目录大小排序显示:

    du -m -d 1 | sort -k1 -n -t' '

  • 磁盘剩余容量:

    df
